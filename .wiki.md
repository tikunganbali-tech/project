# Project Summary
TOKOTANIONLINE is an SEO-driven agricultural e-commerce platform designed to facilitate the online sale of agricultural products while providing educational content for farmers. The platform aims for sustainability, high Google ranking, and organic traffic generation, supported by integrations with WhatsApp and various marketplaces, all managed through a custom CMS. It ensures a smooth shopping experience for users and simplifies content management for administrators.

# Project Module Description
The project comprises several functional modules:
- **Frontend**: Developed with Next.js 14, ensuring responsive design and SEO optimization.
- **Backend**: Built with Node.js and PostgreSQL for robust data management.
- **CMS**: A custom headless CMS for managing products, blogs, and other content.
- **AI Content Automation**: Automates the generation and scheduling of blog content.
- **Checkout System**: Integrates with WhatsApp and various marketplace checkout options.
- **Analytics**: Tracks marketing campaign performance and user interactions.
- **Supabase Integration**: New feature for enhanced database management and connection testing.

# Technology Stack
- **Frontend**: Next.js 14, Tailwind CSS, React 18
- **Backend**: Node.js, PostgreSQL
- **CMS**: Custom headless CMS
- **Authentication**: Supabase for role-based access
- **Deployment**: Vercel or VPS

# Usage
To get started with the project:
1. **Install Dependencies**:
   ```bash
   cd /workspace/shadcn-ui
   pnpm install
   ```
2. **Add Supabase Dependencies**:
   ```bash
   pnpm add @supabase/supabase-js @supabase/ssr
   ```
3. **Build the Project**:
   ```bash
   pnpm run build
   ```
4. **Run the Project**:
   ```bash
   pnpm run start
   ```
